In JavaScript, the `switch` statement is used to evaluate an expression and execute different blocks of code based on the value of that expression. It is an alternative to multiple `if-else` statements, making the code more readable when there are many conditions to check.

Basic Syntax: 


            switch (expression) {
                case value1:
                    // Code to execute if expression === value1
                    break;
                case value2:
                    // Code to execute if expression === value2
                    break;
                default:
                    // Code to execute if no case matches
            }
             
            
             Example: 
             
            let day = 2;
            switch (day) {
                case 1: console.log("Monday"); break;
                case 2: console.log("Tuesday"); break;
                default: console.log("Invalid day");
            }
The `break` statement prevents fall-through, and the `default` case handles unmatched values.

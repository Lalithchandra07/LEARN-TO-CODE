In JavaScript, decision-making is done using conditional statements such as `if`, `else if`, and `else`. These statements evaluate expressions and execute different code blocks based on whether the condition is true or false.

Basic Syntax: 


            if (condition) {
                // Code to execute if condition is true
            } else if (anotherCondition) {
                // Code to execute if the second condition is true
            } else {
                // Code to execute if no condition is true
            }
             
            
            Example:
             
            let age = 20;
            if (age >= 18) {
                console.log("Adult");
            } else {
                console.log("Minor");
            }
The `if` statement checks conditions, and depending on the outcome, it executes specific code blocks.

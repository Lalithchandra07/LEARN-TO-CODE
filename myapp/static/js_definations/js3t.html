Type conversion in JavaScript is the process of converting one data type to another. There are two types of type conversion: implicit (or coercion) and explicit (or casting).

1. Implicit Conversion : JavaScript automatically converts data types when necessary.
let num = "5" + 3; // "53", string concatenation (implicit conversion)

2. Explicit Conversion : You can manually convert types using functions like `Number()`, `String()`, and `Boolean()`.
let str = "123";
let num = Number(str); // Convert string to number
let bool = Boolean(1); // Convert to boolean
JavaScript performs type coercion during operations, but explicit conversion ensures predictable results.

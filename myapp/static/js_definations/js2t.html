In JavaScript, variables are used to store and manage data. Variables can be declared using three keywords: var , let , and const .

- var : The traditional keyword, function-scoped (older).
- let : Block-scoped, recommended for most modern code.
- const : Also block-scoped, used for variables that cannot be reassigned.
Basic Syntax: 


            let name = "Alice";  // Declare with let
            const age = 30;      // Declare with const, value cannot be changed
            var city = "New York"; // Declare with var (older approach)
Variables can hold different types of data like strings, numbers, and objects. let and const are more commonly used in modern JavaScript for better scoping and immutability.

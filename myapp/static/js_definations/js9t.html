In JavaScript, a string is a sequence of characters enclosed in single (`'`), double (`"`), or backticks (`` ` ``). Strings are immutable, meaning their values cannot be changed once created. JavaScript provides several methods to manipulate strings, such as concatenation, searching, and replacing.

Basic Syntax: 


            let str1 = "Hello";
            let str2 = 'World';
            let greeting = `${str1}, ${str2}!`;  // Template literal
             
            
            Common string methods include:
            - `length`: Returns the length of the string.
            - `toUpperCase()`: Converts the string to uppercase.
            - `substring()`: Extracts a part of the string.
            
             Example: 
             
            let name = "JavaScript";
            console.log(name.toUpperCase()); // Output: JAVASCRIPT
Strings are essential for handling text and user input in JavaScript.

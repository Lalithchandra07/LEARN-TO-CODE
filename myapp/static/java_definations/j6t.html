In Java, wrapper classes are used to convert primitive data types into objects. This is necessary because Java collections like `ArrayList` or `HashMap` can only store objects, not primitive types. Wrapper classes also provide utility methods to convert between primitive values and strings or to perform other operations. Each primitive type has a corresponding wrapper class in Java.

Wrapper classes:
- `Integer` for `int`
- `Double` for `double`
- `Character` for `char`
- `Boolean` for `boolean`

Basic Syntax: 


            int x = 10;
            Integer obj = Integer.valueOf(x);  // Convert primitive to wrapper
            int y = obj.intValue();            // Convert wrapper to primitive
Wrapper classes are part of the `java.lang` package and support autoboxing and unboxing, allowing automatic conversion between primitives and objects.

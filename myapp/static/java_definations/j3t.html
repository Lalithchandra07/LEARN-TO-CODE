In Java, logical operators are used to combine multiple boolean expressions or conditions. These operators are essential for decision-making in control structures like if , while , and for . The main logical operators in Java are:

1. && (Logical AND) : Returns true if both conditions are true.
2. || (Logical OR) : Returns true if at least one condition is true.
3. ! (Logical NOT) : Reverses the truth value of a condition.
Basic Syntax: 


            boolean result = (condition1 && condition2); // Logical AND
            boolean result = (condition1 || condition2); // Logical OR
            boolean result = !(condition);               // Logical NOT
             
            
             Example: 
             
            int a = 5, b = 10;
            if (a < b && b > 5) {
                System.out.println("Both conditions are true!");
            }
Logical operators help build complex conditions for controlling program flow.

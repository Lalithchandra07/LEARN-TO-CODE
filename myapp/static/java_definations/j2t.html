In Java, GUI (Graphical User Interface) development is typically done using the Swing or JavaFX libraries. Swing is part of the Java Standard Library, offering a set of lightweight components for building window-based applications. JavaFX, introduced later, provides more advanced features for creating modern UIs, including support for animations and media.

Basic Syntax (Using Swing): 


            import javax.swing.*;
            
            public class SimpleGUI {
                public static void main(String[] args) {
                    JFrame frame = new JFrame("Hello GUI");
                    JButton button = new JButton("Click Me!");
                    frame.add(button);
                    frame.setSize(300, 200);
                    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
                    frame.setVisible(true);
                }
            }
This code creates a simple window with a button labeled "Click Me!" and displays it.

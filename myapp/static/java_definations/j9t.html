Object-Oriented Programming (OOP) in Java is a programming paradigm based on objects and classes. It focuses on encapsulation, inheritance, polymorphism, and abstraction to organize and manage software development. In Java, a class defines the blueprint for objects, and objects are instances of classes.

Basic Concepts:
1. Class : Defines the properties (fields) and behaviors (methods).
2. Object : An instance of a class.
3. Inheritance : A mechanism for creating a new class based on an existing class.
4. Polymorphism : The ability to take many forms (method overriding and overloading).
5. Encapsulation : Hiding data within a class and providing access through methods.


             Example: 
            
            class Car {
                String model;
                void start() {
                    System.out.println("Car is starting");
                }
            }
            
            public class Main {
                public static void main(String[] args) {
                    Car myCar = new Car();  // Creating object
                    myCar.start();
                }
            }
OOP in Java helps create reusable and maintainable code.

In Java, the final keyword is used to define constants, prevent method overriding, and prevent inheritance. It can be applied to variables, methods, and classes:

1. final variable : Once assigned, its value cannot be changed.
2. final method : Cannot be overridden by subclasses.
3. final class : Cannot be subclassed.
Syntax: 


            final data_type variable_name = value;   // Final variable
            
            
             Example: 
            
            final int MAX_VALUE = 100;  // Constant value
            
            class Parent {
                final void display() {  // Final method
                    System.out.println("Parent display");
                }
            }
final provides immutability, ensuring that values, methods, or classes remain unchanged.

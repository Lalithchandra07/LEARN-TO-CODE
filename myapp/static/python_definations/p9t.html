Dictionaries in Python are unordered, mutable collections that store data as key-value pairs. They are defined using curly braces `{}` with the syntax `key: value`. Keys must be unique and immutable (e.g., strings, numbers, or tuples), while values can be of any data type. Dictionaries support operations like adding, updating, deleting, and retrieving values using keys. Common methods include `get()`, `keys()`, `values()`, and `items()`.

Example:


    
    # Creating a dictionary
    person = {"name": "Alice", "age": 25, "city": "New York"}
    
    # Accessing values
    print(person["name"])  # Output: Alice
    print(person.get("age"))  # Output: 25
    
    # Adding and modifying keys
    person["age"] = 26
    person["job"] = "Engineer"
    
    # Removing keys
    del person["city"]
    
    print(person)  # Output: {'name': 'Alice', 'age': 26, 'job': 'Engineer'}
Dictionaries are efficient for lookups and flexible for data representation.

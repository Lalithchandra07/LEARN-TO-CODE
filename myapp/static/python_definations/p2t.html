Type casting in Python refers to converting one data type into another. This is useful when operations require data in specific formats. Python supports two types of type casting: implicit and explicit. Implicit type casting is done automatically by Python, such as converting an integer to a float during an arithmetic operation. Explicit type casting is done manually using built-in functions like int(), float(), str(), etc.

# Implicit casting


x = 10  # Integer

y = 2.5  # Float

result = x + y  # x is automatically converted to float

print(result)  # Output: 12.5



# Explicit casting

z = "15"

print(int(z) + 5)  # Converts string to integer and adds 5; Output: 20

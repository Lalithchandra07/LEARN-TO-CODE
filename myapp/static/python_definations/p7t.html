Lists in Python are ordered, mutable collections that can store elements of different data types, including numbers, strings, and other lists. Defined using square brackets (`[]`), lists allow various operations like adding, removing, and modifying elements. Common methods include `append()`, `extend()`, `insert()`, `remove()`, and `pop()`. Lists also support slicing, indexing, and iteration.

Example:


    
    my_list = [1, 2, 3, "Python", [4, 5]]
    
    # Accessing elements
    print(my_list[0])  # Output: 1
    print(my_list[-1])  # Output: [4, 5]
    
    # Modifying and adding elements
    my_list[1] = 20
    my_list.append(6)
    print(my_list)  # Output: [1, 20, 3, 'Python', [4, 5], 6]
    
    # Removing elements
    my_list.pop(3)
    print(my_list)  # Output: [1, 20, 3, [4, 5], 6]

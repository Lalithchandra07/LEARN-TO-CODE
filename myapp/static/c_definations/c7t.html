In C, the `switch` statement is a multi-way decision-making construct that executes one code block out of many based on the value of an expression. It is an alternative to using multiple `if-else` statements and improves readability when there are many conditions.

Syntax: 


            switch (expression) {
                case value1:
                    // Code to execute
                    break;
                case value2:
                    // Code to execute
                    break;
                 
                default:
                    // Code to execute if no cases match
            }
Example: 

            int day = 2;
            switch (day) {
                case 1: printf("Monday"); break;
                case 2: printf("Tuesday"); break;
                default: printf("Invalid day");
            }
The `break` statement exits the `switch`, preventing fall-through. The `default` case is optional and handles unmatched values.

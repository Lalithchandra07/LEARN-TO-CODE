Decision-making in C allows the program to choose between different actions based on conditions. It evaluates expressions that return true or false and executes specific code accordingly.

Types of Decision-Making Statements:
1. if Statement : Executes code if the condition is true.
2. if-else : Executes one block if true, another if false.
3. else if Ladder : Tests multiple conditions sequentially.
4. switch Statement : Selects code to execute based on a variable's value.

Syntax : 

  if  Statement: 
                c
               if (condition) {
                   // Code to execute
               }
These statements make programs logical and adaptable.

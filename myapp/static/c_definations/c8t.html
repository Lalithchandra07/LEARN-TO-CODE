An array in C is a collection of elements of the same data type, stored in contiguous memory locations. Arrays allow storing multiple values under a single name, making data management efficient. Each element is accessed using an index, starting from 0.

Syntax: 

data_type array_name[size];
Example: 

            int numbers[5] = {1, 2, 3, 4, 5}; // Integer array
            char name[10] = "Alice";          // Character array (string)
Accessing Elements: 

            printf("%d", numbers[2]); // Outputs 3
Arrays can be single-dimensional, multi-dimensional (like matrices), or character arrays (strings). They are widely used for data storage and manipulation in C programs.

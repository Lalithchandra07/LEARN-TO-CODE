A 2D array in C is an array of arrays, allowing the storage of data in a matrix-like structure. It is used to represent tables, grids, or other data with rows and columns. Each element is accessed using two indices: one for the row and one for the column.

Syntax: 


            data_type array_name[row_size][column_size];
Example: 

            int matrix[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
Accessing Elements: 

            printf("%d", matrix[1][2]); // Outputs 6
2D arrays can be used for more complex data manipulation like matrices in mathematical computations or grids in games.

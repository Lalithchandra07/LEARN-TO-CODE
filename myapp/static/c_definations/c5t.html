Arithmetic operators in C are used to perform basic mathematical operations on numeric data types. They include addition, subtraction, multiplication, division, and modulus.

List of Arithmetic Operators:
1. `+` (Addition): Adds two numbers.
2. `-` (Subtraction): Subtracts one number from another.
3. `*` (Multiplication): Multiplies two numbers.
4. `/` (Division): Divides one number by another.
5. `%` (Modulus): Returns the remainder of division.

Syntax: 

result = operand1 operator operand2;

Example: 

            int a = 10, b = 3;
            int sum = a + b;      // 13
            int difference = a - b; // 7
            int product = a * b;  // 30
            int quotient = a / b; // 3
            int remainder = a % b; // 1
These operators simplify mathematical computations in programs.

Data types in C define the type and size of data a variable can hold, ensuring efficient memory usage and proper operations. They are broadly categorized into:

1. Primary Data Types : `int`, `float`, `char`, `double`.
2. Derived Data Types : Arrays, Pointers, Functions.
3. Enumeration Types : `enum`. 4. User-defined Types : `struct`, `union`, `typedef`. Syntax: data_type variable_name; Example: 


            int age = 20;       // Integer
            float pi = 3.14;    // Floating-point
            char grade = 'A';   // Character
            double large = 10.56; // Double
Each type has a specific size and range, determined by the system architecture, enabling precise data handling and processing.

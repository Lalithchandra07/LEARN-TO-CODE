A nested loop in C++ is a loop inside another loop, used when you need to perform repetitive tasks within another repetitive task. Typically, outer loops control the number of iterations for the inner loop. Nested loops are commonly used in situations like multi-dimensional array traversal or matrix operations.

Syntax: 


            for (int i = 0; i < n; i++) {
                for (int j = 0; j < m; j++) {
                    // Code to execute
                }
            }
Example: 

            for (int i = 1; i <= 3; i++) {
                for (int j = 1; j <= 3; j++) {
                    cout << i << " " << j << endl;
                }
            }
This prints pairs of `i` and `j`, demonstrating a 2D iteration. Nested loops can be extended to more levels for more complex iterations.

In C++, `typedef` is used to create an alias or shorthand for an existing data type, improving code readability and maintainability. It allows you to define a new name for an existing type, making complex type declarations easier to work with. `typedef` is especially useful with pointers, structs, or function pointers.

Syntax: 


            typedef existing_type new_name;
Example: 

            typedef unsigned long ulong;   // Alias for unsigned long
            typedef int (*func_ptr)(int);  // Function pointer alias
            
            int main() {
                ulong num = 100000;  // Using the alias
                return 0;
            }
Here, `ulong` is an alias for `unsigned long`, and `func_ptr` is a type for function pointers.

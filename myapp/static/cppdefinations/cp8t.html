In C++, the `switch` statement is used to execute one of many code blocks based on the value of an expression. It simplifies decision-making when there are multiple conditions to check, offering a cleaner alternative to multiple `if-else` statements. The expression in the `switch` is typically an integer or character.

Syntax: 


            switch (expression) {
                case value1:
                    // Code to execute if expression == value1
                    break;
                case value2:
                    // Code to execute if expression == value2
                    break;
                default:
                    // Code to execute if no case matches
            }
Example: 

            int day = 2;
            switch (day) {
                case 1: cout << "Monday"; break;
                case 2: cout << "Tuesday"; break;
                default: cout << "Invalid day";
            }
The `break` statement prevents fall-through, while the `default` case handles unmatched values.

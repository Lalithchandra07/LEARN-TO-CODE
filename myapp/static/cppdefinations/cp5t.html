Logical operators in C++ are used to perform logical operations on boolean values, typically in decision-making statements like `if` or `while`. They allow combining multiple conditions into a single expression. The three main logical operators are:

1. `&&` (Logical AND) : Returns true if both conditions are true. 2. `||` (Logical OR) : Returns true if at least one condition is true. 3. `!` (Logical NOT) : Reverses the truth value of a condition. Syntax: 


            condition1 && condition2
            condition1 || condition2
            !condition
Example: 

            if (x > 10 && y < 20) {
                cout << "Both conditions are true!" << endl;
            }
Logical operators help build complex conditions in C++ programs.

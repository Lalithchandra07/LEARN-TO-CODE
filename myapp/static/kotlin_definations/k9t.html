In Kotlin, arrays are collections of fixed-size elements of the same type. Arrays are created using the `arrayOf()` function or specialized functions like `IntArray()` for primitives. They support indexed access, starting from 0.

Basic Syntax: 


            val numbers = arrayOf(1, 2, 3, 4)  // Array of integers
            val names = arrayOf("Alice", "Bob")  // Array of strings
            
            println(numbers[0])  // Access element at index 0, Output: 1
            numbers[1] = 10      // Modify element at index 1
            println(numbers[1])  // Output: 10
            
            
             Loop Through an Array: 
            kotlin
            for (num in numbers) {
                println(num)
            }
Kotlin arrays provide methods like `size`, `sort()`, and `contains()` for easy manipulation and queries. They are versatile and commonly used for storing collections of data.

In Kotlin, a `while` loop is used to repeatedly execute a block of code as long as a specified condition is true. It checks the condition before executing the loop body, making it an entry-controlled loop.

Syntax: 


            while (condition) {
                // Code to execute
            }
Example: 

            var count = 1
            while (count <= 5) {
                println("Count: $count")
                count++  // Increment the counter
            }
            
            
             Output: 
            
            Count: 1
            Count: 2
            Count: 3
            Count: 4
            Count: 5
The `while` loop is commonly used for tasks where the number of iterations isn't predetermined but depends on a condition being met.

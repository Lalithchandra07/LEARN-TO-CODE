In Kotlin, strings are sequences of characters enclosed in double quotes ( " " ). They are immutable, meaning their value cannot be changed after creation. Kotlin offers powerful string manipulation features, including string templates and methods for formatting and operations.

String Templates : Use $ for variable interpolation or ${} for expressions.
Example: 


            val name = "Kotlin"
            val greeting = "Hello, $name!"
            println(greeting)  // Output: Hello, Kotlin
Common Methods :
- length : Returns string length.
- uppercase() : Converts to uppercase.
- substring() : Extracts part of a string.

             Example: 
            kotlin
            val str = "Kotlin"
            println(str.length)         // Output: 6
            println(str.uppercase())    // Output: KOTLIN
            println(str.substring(1, 4)) // Output: otl
Strings in Kotlin are versatile and easy to work with.

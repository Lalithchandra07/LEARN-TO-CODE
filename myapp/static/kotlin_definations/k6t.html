In Kotlin, logical operators are used to evaluate boolean expressions and combine multiple conditions. The primary logical operators are:

1. && (AND) : Returns true if both conditions are true.
2. || (OR) : Returns true if at least one condition is true.
3. ! (NOT) : Reverses the boolean value ( true to false and vice versa).
Basic Syntax: 


            val a = true
            val b = false
            
            println(a && b)  // Output: false (both must be true)
            println(a || b)  // Output: true (one is true)
            println(!a)      // Output: false (negation of true)
Logical operators are commonly used in decision-making structures like if statements to control program flow based on conditions.

In Kotlin, a List is an ordered collection of elements. Kotlin offers two types of lists:

1. Immutable List (`listOf`) : Read-only; elements cannot be modified.
2. Mutable List (`mutableListOf`) : Allows adding, removing, or updating elements.
Basic Syntax: 


            // Immutable list
            val fruits = listOf("Apple", "Banana", "Cherry")
            println(fruits[0])  // Access element by index, Output: Apple
            
            // Mutable list
            val numbers = mutableListOf(1, 2, 3)
            numbers.add(4)       // Add element
            numbers[1] = 10      // Update element
            numbers.remove(3)    // Remove element
            println(numbers)     // Output: [1, 10, 4]
Lists support iteration, sorting, filtering, and many useful operations. They are versatile for managing dynamic collections of data.

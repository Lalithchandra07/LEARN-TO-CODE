In Kotlin, arithmetic operators are used to perform basic mathematical operations. These operators work on numeric types like Int , Float , Double , etc.

Arithmetic Operators:
1. + (Addition): Adds two values.
2. - (Subtraction): Subtracts the second value from the first.
3. * (Multiplication): Multiplies two values.
4. / (Division): Divides the first value by the second.
5. % (Modulus): Returns the remainder of division.
Basic Syntax: 


            val a = 10
            val b = 3
            
            println(a + b)  // Output: 13
            println(a - b)  // Output: 7
            println(a * b)  // Output: 30
            println(a / b)  // Output: 3
            println(a % b)  // Output: 1
These operators are straightforward and used in calculations throughout Kotlin programming.

In Kotlin, decision-making statements allow you to execute code blocks based on conditions. The primary statements are if , else if , and else .

1. if Statement: Executes code if the condition is true.
2. else Statement: Executes code if the if condition is false.
3. else if Statement: Tests multiple conditions.
Basic Syntax:


val number = 10

if (number > 0) {
    println("Positive number")
} else if (number == 0) {
    println("Zero")
} else {
    println("Negative number")
}
 

Output:
 
Positive number
Kotlin's if can also be used as an expression to return values: kotlin val result = if (number > 0) "Positive" else "Negative"

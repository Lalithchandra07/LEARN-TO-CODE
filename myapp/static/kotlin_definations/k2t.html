In Kotlin, variables are declared using either var or val .

- var : A mutable variable whose value can change.
- val : An immutable variable, similar to a constant, whose value cannot be reassigned once set.

Kotlin uses type inference but allows explicit type declaration.

Basic Syntax: 


            var age: Int = 25  // Mutable integer variable
            val name: String = "Kotlin"  // Immutable string variable
            
            age = 30  // Allowed because  var  is mutable
            // name = "Java"  // Error:  val  cannot be reassigned
            
            // Type inference (no explicit type required)
            var city = "New York"
            val year = 2024
Using var and val ensures clear and concise variable management in Kotlin.

WHAT IS CODING

Coding is the process of creating instructions for computers to follow. These instructions, written in specific programming languages like Python or JavaScript, tell the computer what tasks to perform, such as displaying a webpage or calculating numbers. Coding is how we communicate with computers to make them do things—whether it’s a simple task, like showing text on a screen, or a complex one, like controlling a robot. Essentially, coding transforms human ideas into a language computers understand, allowing us to build apps, websites, games, and much more.

WHAT ARE CODING LANGUAGES

A coding language, or programming language, is a set of rules and syntax used to write instructions for computers. These languages allow us to create software, applications, websites, and more by translating human commands into a form a computer can process.
Types of coding languages:

1. Low-level languages: Close to machine code, which computers directly execute. Examples include Assembly Language and Machine Code.
2. High-level languages: More abstract and human-readable, often used for software development. Examples are Python, Java, and JavaScript.
3. Markup languages: Used for structuring and presenting data on websites. Examples include HTML and XML.
4. Query languages: Designed to retrieve data from databases. An example is SQL (Structured Query Language).
Each type serves different purposes, from web development to database management.

WHAT IS SYNTAX
In simple terms, syntax is the set of rules that defines how to write code in a programming language. It’s like grammar in a spoken language; syntax dictates the correct arrangement of words, symbols, and expressions to create instructions a computer can understand. For example, in many languages, you end each line with a semicolon (`;`), use `{}` to group code, and follow specific keywords for tasks (like `if`, `else`, `for`). If you don’t follow syntax rules, the program won’t run correctly, because the computer won’t be able to interpret your instructions.
